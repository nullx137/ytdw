YT Downloader — скачивание видео и аудио с YouTube

КАК ПОЛЬЗОВАТЬСЯ:
1. Запустите "YT Downloader.exe"
2. Вставьте ссылку на YouTube видео
3. Нажмите "Get Info"
4. Выберите качество из списка
5. Нажмите "Download"

НОВЫЕ ВОЗМОЖНОСТИ:
- Отмена загрузки (кнопка Cancel)
- Тёмная и светлая тема (кнопка Dark/Light)
- Субтитры: отметьте "Download subtitles", укажите языки (en,ru)
- Выбор битрейта MP3 (128-320 kbps)
- Контекстное меню в логе (правая кнопка мыши)
- Открытие папки после скачивания

MP3 (ТОЛЬКО АУДИО):
1. Отметьте "Download MP3 (audio only)"
2. Выберите битрейт
3. Нажмите Download
4. ВАЖНО: требуется установленный ffmpeg

УСТАНОВКА FFMPEG (нужен только для MP3):
Способ 1 — winget (рекомендуется):
  winget install ffmpeg
Способ 2 — вручную:
  Скачайте с https://ffmpeg.org/download.html
  Добавьте папку "bin" в переменную PATH

СБОРКА ИЗ ИСХОДНИКОВ:
  Запустите build_exe.bat (требуется Python 3.8+)

СИСТЕМНЫЕ ТРЕБОВАНИЯ:
- Windows 7 или новее
- yt-dlp встроен в .exe
- ffmpeg нужен ТОЛЬКО для конвертации в MP3
